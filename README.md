## 開発背景
### 作者のバックグラウンド：「太古のロマン」
  
作者をざっくり表現すると、  
  
- 幼少期〜高校時代＝恐竜好き

という人間です。  
大学、大学院生時代は、地質系の学科にてフィールドワークを行ってきました。地層はとても美しいです。自然は不均質な形や性質をしています（例：木は決まった形をしていない）。しかし、地層はその不均質に対して、直線や曲線を岩壁に描いていています。これは地球が46億年かけてつくった壮大なアートです。

### 世の中の課題の発見：「野外写真がもっとカジュアルになれば・・・」
前述までの説明をお読み頂いて、「地質学ってとてもニッチな学問だな。」とお思いになったのではないのでしょうか？私もそう思います。日本全体で比較しても、地質系学科は非常に少なく、地質系分野を対象としたコンテンツやサービスで有名なものはあまり無いのが現状です。  
私が常日頃から感じている問題点は次の２点です。
①フィールドワークをしている人は写真をたくさん所持しており、写真一枚一枚に、努力・思い出・結果・達成感が詰まっている。  
②写真に使用は、主にレポート・ゼミ発表・卒業発表の３場面（かしこまったシーン）しか無く、その実施頻度も少ない（月１回〜半年１回）。  
  
私は、上記の問題点に対し、もっとカジュアルにフィールドワークをアウトプットできて、もっと気軽に共有して楽しむべきとだと思っています。
  

## ペルソナ
- 男性  
本アプリが対象としているフィールドワークを行なっている学生人口は、（体感的に）男性が過半数を占めている傾向があります。現に、作者在学中も、同学年で生徒５０人中、女性は９人でした。  
- １８〜３０代・大学生〜大学院生（博士）  
「①流行に敏感」で、最も自分の行動を「②発信する時間と環境が担保されている状態」なためです。  

- 上記を踏まえた具体的なユーザーストーリー
某地方大学自然科学系学科在学3回生で、授業・研究で月1〜5回のフィールドワークを課されています。写真はこれまでの実習、研究活動でたくさん撮影しており、パソコン内に大量に保管しています。「面白いものを見つけた！」と同学科の友人へ共有したいと思っています。しかし、ツイッターやインスタグラムでは、①分野外の他愛もない投稿に埋もれてしまい、投稿が見て欲しい人に届かない。②そもそも分野外の人に見られる事が恥ずかしい。という問題を抱えていました。  
また本人は、学会や他大学の同じ分野の学生と知り合いたいとも思っています。

### 新規登録・ログイン画面

未ログイン状態で画面右上の新規登録ボタンを押すと、新規登録画面へ遷移します。必要情報を入力し、最下部の登録ボタン（緑色）を押すことによって登録完了となります（DEMO画像5, 6）。  
登録情報は以下の通りです。（*　は必須）

- 名前 or ニックネーム*
- プロフィール画像（任意）  
- メールアドレス*
- パスワード*（確認用含め2回）
- 専門科目（選択）
- 現在の研究・興味のあるテーマ
- 自己紹介

新規登録完了後、ログインボタンよりログインが可能です。  
ログイン時の入力情報は以下の通りです（DEMO画像7）。  

- メールアドレス
- パスワード

## 本アプリへの思い
私は小さな頃から恐竜が好きでした。  
両親にもその理解があり、博物館や大恐竜博などのイベントによく連れて行ってもらっていました。  
しかし、高校は私立。大学受験では浪人。  
夢のためにと、両親は私にたくさんのお金をかけてくれました。  
両親には金銭的なことも含め様々な負担をかけたと思っています。  
大学と大学院では地質学を学びました。  
特に、地層の形成過程をフィールドワークでサイエンスする「堆積学」という学問にハマりました。  
地層は「不均質な自然界に存在する直線や曲線」です。  
これは「地球が46億年の歳月をかけてかけて作った唯一無二の芸術」だと思っています。  
また、大学、大学院で尊敬する2人の恩師に恵まれ、人間としても、地質屋としても育ててもらいました。  
この間、仲間もたくさんできました。  
同じ学問を楽しんだり、時には歯を食いしばって、共に課題をこなして頑張ったり、大学院受験で上京して他大学から来たよそ者の私を優しく受け入れてくれました。  
現職では、理解のある上司と同僚、先輩の元で好きな地層に対する仕事ができています。  
幸せそのものです。  
今の私は、間違いなく、今まで関わってきてくれた周囲の人たちのおかげでできています。  
しかし私はエンジニアになろうとしています。  
私の行為は、見方によってはいままで積み上げてきたものを捨てて未知の分野に飛び込むということです。  
地質学が嫌いになったわけではありません。  
私は、自分の表現能力と可能性を広げたいと思っています。  
プログラミングはどんな分野、業界にも通用する「世界を包括しうる技術」だと思い、魅力に感じています。  
このFieldwork Recorderは、そんな新しい一歩に当たる作品です。  
覚えたてのコードで今までの自分を精一杯表現しました。  
少しでも多くの人に触っていただける作品にしたいです。
